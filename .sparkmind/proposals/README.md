# Proposals — Indeks

Ketika AI Engineer menemukan requirement yang tidak jelas atau keputusan yang
membutuhkan persetujuan Founder/CTO, proposal dibuat di sini
(pakai `../templates/proposal.md`), format: `YYYY-MM-DD-judul.md`.

Setelah diputuskan:
- Jika menghasilkan keputusan arsitektur → buat ADR di `../decisions/`.
- Update status proposal (✅ Disetujui / ❌ Ditolak) — jangan dihapus.
